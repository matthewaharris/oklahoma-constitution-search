About This Project

What is this?

The Oklahoma Constitution Search is an AI-powered semantic search engine that allows you to search the Oklahoma State Constitution using natural language queries.

How does it work?

This system uses cutting-edge AI technology to understand the meaning behind your searches:

  • Semantic Understanding: Unlike traditional keyword search, this system understands the meaning and context of your query.
  • Vector Embeddings: Each section of the Oklahoma Constitution is converted into a mathematical representation (vector) using OpenAI's embedding model.
  • Similarity Search: Your search query is also converted into a vector, and the system finds the most semantically similar sections using Pinecone vector database.
  • Relevance Scoring: Results are ranked by relevance, with scores showing how well each section matches your query.

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python, Flask
  • Embeddings: OpenAI text-embedding-ada-002
  • Vector Database: Pinecone
  • Data Storage: Supabase (PostgreSQL)
  • Data Source: Oklahoma State Courts Network (OSCN)

Features

  • Natural language search
  • AI-powered semantic understanding
  • Relevance scoring for each result
  • Fast, sub-second response times
  • Mobile-friendly interface

Use Cases

  • Citizens researching their constitutional rights
  • Students studying Oklahoma government and law
  • Legal professionals conducting research
  • Journalists and researchers
